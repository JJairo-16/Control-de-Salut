# Control de Salut - Jairo Linares

## Què és?

En aquest programa es busca controlar la salut general física de les persones.

## Opcions

1. Introduir dades
</br>Obtenir per consola i válidar les següents dades:
    - Nom complet
    - Edat
    - Pes (Kg)
    - Alçada (m)
